Node.js API Endpoints for Assignment 8

Because this assignment requires running the server locally, I have only provided documentation of how it works below.

The app.js file has been submitted on Canvas. Other files can be found in the GitHub repository.


Initial Set up

GET /hello

curl http://localhost:8000/hello

Returns: "Hello World!"

Exercise 1

GET /math/circle/:r

Takes the radius as a URL parameter and calculates circle area and circumference and returns solution as JSON

curl http://localhost:8000/math/circle/5

Returns: {"area": 78.54, "circumference": 31.42} on the page.

Exercise 2

GET /hello/name

Takes query parameters in the URL (first and last)

curl http://localhost:8000/hello/name?first=John&last=Doe

Returns: "Hello John Doe" on the page.
